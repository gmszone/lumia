##Mac OS webkit##

将app复制到Application，接着把下面的alias添加到~/.bash_profile

    # alias to nw 
    alias nw="/Applications/node-webkit.app/Contents/MacOS/node-webkit"

``source ~/.bash_profile``

    nw app.nw
